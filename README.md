## 🧭 Introduction

Hello! We’re a team of three students – **Rajni**, **Saddam**, and **Simranjeet** – working on this academic project titled **Smart Job Tracker**, an **AI-powered WordPress web application** built to help individuals manage and optimize their job search process effectively.

This application is designed for anyone tired of juggling multiple spreadsheets or lost bookmarks — now everything is in one centralized, intelligent platform.

---

## ❗ Problem Statement

Most job seekers rely on simple tools like spreadsheets, sticky notes, or browser bookmarks to track job applications — tools that lack structure, automation, and intelligent reminders.

---

## 🎯 Our Goal

To create a **centralized WordPress-based solution** that helps users:

- Track job applications
- Receive interview and follow-up reminders
- Auto-generate resumes and cover letters using AI
- Streamline the entire job-hunting experience in a modern, responsive UI

---

## 🔑 Core Features

✅ **Custom Job Dashboard**  
   Built using **WP Job Manager**, allowing users to log jobs, track progress, and categorize applications (Applied, Interview, Offer, etc.)

✅ **Automated Job Import**  
   Integrated with **Jobs Aggregator** and **Indeed** to fetch real-time job listings

✅ **AI Resume & Cover Letter Generator**  
   Uses **OpenAI GPT-4 API** to create tailored documents based on job descriptions and user profiles

✅ **Smart Reminders**  
   Users can set follow-up reminders and interview alerts

✅ **Responsive Interface**  
   Built using **Elementor** and custom lightweight CSS for mobile-friendly layouts

## 🔁 Workflow & System Design

- Manual and automated job entry supported
- CPT (Custom Post Types) used for managing structured job data
- AI generation triggered from forms via API hooks
- Custom Elementor widgets used to structure frontend dashboard
- External job feeds regularly synced to provide fresh listings
- Clean UI layout for dashboard, resume preview, and calendar integration
